Try to catch the red point!

Instructions

  • Move the blue point using arrow keys (↑←↓→)
  • In two-player mode, red point moves with WASD keys
  • Game Modes:
  • Offense: Try to catch the escaping red point
  • Defense: Try to escape from the chasing red point
  • Two Player: Take turns moving points
  • After each move, a random connection disappears
  • Red point always moves first
  • If points get separated with no path between them:
  • Offense Mode: Red Wins
  • Defense Mode: Blue Wins

Strategy Tip: Plan your moves carefully - the path you break might be your own escape route!
