Catch the Point

Try to catch the red point!

How to Play

Offense Mode

• You control the blue point with arrow keys

• Your goal: Catch the red point by moving next to it

• Strategy: Try to trap red by cutting off escape routes

Defense Mode

• You control the red point with arrow keys

• Your goal: Keep escaping from the blue point

• Strategy: Stay in open areas with multiple escape paths

Game Rules

  • After each move, a random path disappears
  • Game ends when points connect or are separated
  • Press Enter anytime to reset the game
